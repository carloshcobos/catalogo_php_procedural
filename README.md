# catalogo_php_procedural

Este repositorio contiene un proyecto en PHP procedural con mysqli embebido en HTML, desarrollado sin usar MVC ni buenas prácticas modernas. El objetivo de publicar este código es consulta y referencia educativa, no su uso en producción.

Soy consciente de que el código es obsoleto en su forma y estructura, y precisamente lo comparto como ejemplo para aprender a:
- Entender cómo se construían aplicaciones web con PHP tradicional.
- Identificar limitaciones y riesgos de seguridad en el enfoque procedural.
- Tomar este punto de partida para migrar a una arquitectura más actual (MVC, PDO, frameworks, etc.).

ADVERTENCIA: No usar este código directamente en entornos reales o productivos.
